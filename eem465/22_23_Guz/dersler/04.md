# EEM-465 Gömülü Sistemler

## 2022-2023 Güz Dönemi

### RCC_AHB1 yazmacı için makro tanımlama

```C
#define RCC_AHB1ENR (*((int*)0x40023830))
int main(){
  RCC_AHB1ENR |= 1<<3;
}
```

### LED yakma programı
```C
#define RCC_AHB1ENR (*((int*)0x40023830))
#define GPIOD_MODER (*((int*)0x40020C00))
#define GPIOD_ODR (*((int*)0x40020C14))
int main(){
  RCC_AHB1ENR |= 1<<3;
  GPIOD_MODER |= 1<<30;
  GPIOD_ODR |= 1<<15;  
}
```
